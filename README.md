# Pizza-Sales_Project_PowerBI
Hey all!!



Built an end-to-end Pizza Sales Analytics Dashboard using Excel and Power BI to transform raw transactional data into actionable business insights.



🔍 Analysis performed:

✅ Monthly order trend analysis using stacked bar chart

✅ Top 5 best-selling pizzas identification using stacked column chart

✅ Bottom 5 low-performing pizzas analysis using stacked column chart

✅ Sales contribution by pizza category using pie chart

✅ Customer preference analysis by pizza size using donut chart

✅ Quarterly performance filtering using interactive slicers

✅ Total pizzas sold based on pizza category using funnel chart



🛠️ Tools & Skills Used:

• Excel – Initial data handling and validation

• Power BI – Data loading from Excel, cleaning, dashboard development & data visualization, DAX measures



📊 Key insights from the project:

• Total Revenue Generated: 817K+

• Total Pizzas Sold: 49.5K

• Average Order Value: 38.31

• Average Pizzas per Order: 2.32



💡 Project Insights:

• Classic Pizza category recorded the highest overall sales contribution (~27%)

• Large-sized pizzas dominated customer demand with nearly 46% sales share

• The Classic Deluxe Pizza emerged as the top-selling pizza overall

• Lower-performing pizza like The Brie Carre Pizza showed significantly lower customer demand

• Monthly sales remained relatively stable, with stronger performance during May and July
